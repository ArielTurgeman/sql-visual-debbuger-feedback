# SQL Visual Debugger - Feedback & Issues

This repository is the public issue tracker and feedback home for **SQL Visual Debugger**.

SQL Visual Debugger is a VS Code extension that helps you visually debug supported SQL `SELECT` queries. It shows intermediate results, row changes, JOIN previews, grouping breakdowns, and more so you can understand how a query produced its final result.

The extension source code is private. This public repository is only for:

- bug reports
- feature requests
- questions
- screenshots
- marketplace assets
- user feedback

## Links

- [Open a new issue](https://github.com/ArielTurgeman/sql-visual-debugger-feedback/issues/new/choose)

## Before Opening An Issue

Please include as much relevant context as you can:

- database type: MySQL, PostgreSQL, SQL Server, or SQLite
- SQL query, if it is safe to share
- expected result
- actual result
- error message
- screenshots
- SQL Visual Debugger extension version

Please do **not** include passwords, secrets, production credentials, private hostnames, customer data, or other sensitive information.
