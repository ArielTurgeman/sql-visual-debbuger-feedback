# SQL Visual Debugger Feedback

This repository is for feedback, bug reports, and feature requests for SQL Visual Debugger.

SQL Visual Debugger is a VS Code extension that helps you debug SQL visually, clause by clause.

## Install the extension

Install SQL Visual Debugger from the Visual Studio Marketplace:

[SQL Visual Debugger on Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=arieldev.sql-visual-debugger)

## Send feedback

Found a bug, confusing behavior, or something that could be better?

[Open a new issue](https://github.com/ArielTurgeman/sql-visual-debbuger-feedback/issues/new/choose)

## What to include

For bug reports, please include:

- What database you were using: MySQL, PostgreSQL, SQL Server, or SQLite
- What you clicked before the problem happened
- What you expected to happen
- What actually happened
- A screenshot if it helps

Please do not include passwords, private database details, production data, or sensitive SQL.
